# Observability Stack

Demonstration of backend observability best practices.

## Overview
Showcases logging, metrics, and tracing integration for backend services.

## Features
- Structured logging
- Metrics collection
- Distributed tracing
- Health & readiness checks

## Tech Stack
- Logging: Zap / Zerolog
- Metrics: Prometheus
- Tracing: OpenTelemetry

## Architecture
- Instrumented services
- Centralized observability pipeline

## Notes
Focused on production readiness.
